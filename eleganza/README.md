# Eleganza – Modern E-Commerce Website 

Eleganza is a modern, responsive e-commerce web application built using **Next.js**, **React**, and **Tailwind CSS**.  
The project focuses on clean UI, scalability, and real-world e-commerce features suitable for academic and portfolio use.

---

##  Tech Stack

- **Next.js** – React framework for production-ready apps
- **React** – Component-based UI development
- **Tailwind CSS** – Utility-first styling
- **JavaScript (ES6+)**
- **Node.js & npm**

---

##  Features

- Responsive and modern UI
- Product listing and detail pages
- User-friendly navigation
- Scalable project structure
- Ready for backend & payment integration
